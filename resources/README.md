## 如何使用

下载 [我的 BAB 规则文件](https://raw.githubusercontent.com/suliveevil/BetterAndBetter/master/resources/BAB.plist), 在 BetterAndBetter 偏好设置-通用 里点击导入规则即可。

## 通用

-[x] 开机时自启动

-[x] 提醒小贴士

-[x] 显示绘图轨迹
